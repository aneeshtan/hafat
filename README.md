# HAFAT WordPress Theme

This folder is a native WordPress theme version of the HAFAT website.

## Install

1. Copy this repository folder into your WordPress installation under `wp-content/themes/hafat/`.
2. In WordPress Admin, go to `Appearance > Themes`.
3. Activate the `HAFAT` theme.
4. Optional: create menus under `Appearance > Menus` and assign them to `Primary Navigation` and `Footer Navigation`.
5. Set the homepage to use the front page template by going to `Settings > Reading` and selecting a static homepage if needed.

## Notes

- Assets, CSS, and JavaScript are enqueued through `functions.php`.
- The current homepage content is in `front-page.php`.
- The theme preserves the static website design and interactions as native WordPress templates.
